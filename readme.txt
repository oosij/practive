주로 자연어처리와 언어 모델을 연구했으며, 추가적인 관심사로 이미지가 있습니다.

텍스트와 이미지를 멀티 모달 방식의 하이브리드 모델에 현재 관심을 두고 있고, 이미지는 CNN를 비롯해서 GAN 등 study 중에 있습니다.

자연어처리로는, 뉴스 수집, 파싱, 텍스트 분석 등을 비롯해서 언어 모델과 미세조정 등을 해왔으며 최근에는 생성 분야에도 흥미를 가지고 있습니다. 

가장 큰 현재의 진행 관심사는 특정 도메인에 특화된 사전 학습 언어 모델을 직접 구축한 후, 이를 기반으로 다양한 Task에 적용함으로써 성능을 입증하고 
최종적으로 현재 이슈화되고 있는 ChatGPT와 같이 대화형 생성 모델을 만드는 것입니다.

그 다음적으로 가능하다면, 이미지 데이터도 활용해서 Text to image or Image to text 와 같은 멀티 모달 방식의 하이브리드 모델을 구축하고자 합니다.
